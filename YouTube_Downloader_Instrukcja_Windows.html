🚀 Wprowadzenie — co potrafi skrypt

  • 📥 Pobiera filmy z YouTube jako MP4 (najlepsza jakość) lub MP3 (samo audio)
  • 📂 Działa też na plikach z dysku (parametr --file) — bez pobierania
  • 🤖 Generuje napisy lokalnie, za pomocą Whisper AI (OpenAI)
  • 🔁 Tłumaczy napisy na dowolny język (np. film po portugalsku → napisy po polsku) przez Google Translate
  • 🌐 Sam wykrywa język oryginału (tryb --source-lang auto), próbkując kilka momentów nagrania
  • 🛡️ Odporne pobieranie — sam instaluje/aktualizuje yt-dlp, próbuje wielu strategii, wznawia przerwane transfery, diagnozuje błędy
  • ⚡ Obsługuje GPU (NVIDIA/CUDA) — wielokrotnie szybsza transkrypcja
  • 🎯 Dba o jakość — parametry ograniczające „halucynacje" Whispera i opcja podpowiedzi kontekstowej (--prompt)
Jak to działa w skrócie? Whisper zamienia mowę na tekst w języku oryginału (transkrypcja). Jeśli włączysz tłumaczenie, każdy segment napisów jest dodatkowo przekładany na język docelowy. Zapisywane są obie wersje: oryginał i tłumaczenie.

🛠️ Instalacja krok po kroku

1

Zainstaluj Python

Pobierz z https://www.python.org/downloads/ i uruchom instalator.

WAŻNE: zaznacz „Add Python to PATH" na dole okna instalatora!

Sprawdź w PowerShell (Win + R → powershell → Enter):

python --version

Powinno pokazać np. Python 3.13.x. Jeśli python nie działa, spróbuj py.

2

Utwórz folder i środowisko wirtualne (venv)

cd C:\ mkdir yt_downloader cd yt_downloader python -m venv .venv .\.venv\Scripts\activate

Po aktywacji zobaczysz (.venv) przed znakiem zachęty. Venv izoluje pakiety od reszty systemu.

Środowisko aktywujesz za każdym razem przed użyciem skryptu: .\.venv\Scripts\activate
3

Zapisz skrypt

Skopiuj kod z sekcji „Kod skryptu" (na końcu) do pliku youtube_downloader.py w folderze projektu.

4

Zaktualizuj pip i zainstaluj pakiety

python -m pip install --upgrade pip pip install -U yt-dlp pip install -U openai-whisper pip install -U deep-translator
  • yt-dlp — pobieranie z YouTube (skrypt potrafi też zainstalować/zaktualizować je sam)
  • openai-whisper — transkrypcja mowy (instaluje też PyTorch; może potrwać kilka minut)
  • deep-translator — tłumaczenie napisów (Google Translate, bez klucza API)
5

Zainstaluj ffmpeg

ffmpeg jest wymagany do przetwarzania audio/wideo.

  1. Pobierz z https://www.gyan.dev/ffmpeg/builds/ → „ffmpeg-release-essentials.zip"
  2. Rozpakuj do C:\ffmpeg\ (tak, aby istniał C:\ffmpeg\bin\ffmpeg.exe)
  3. Dodaj C:\ffmpeg\bin do zmiennej PATH: Win + R → sysdm.cpl → „Zaawansowane" → „Zmienne środowiskowe" → edytuj „Path" → „Nowy" → wpisz ścieżkę
  4. Zamknij i otwórz ponownie PowerShell, sprawdź: ffmpeg -version

⚡ Przyspieszenie na karcie NVIDIA (GPU)

Domyślnie PyTorch działa na procesorze (CPU). Duże modele (medium, large-v3, turbo) są wtedy wolne — długi film może liczyć się nawet godzinami. Jeśli masz kartę NVIDIA, zainstaluj wersję PyTorch z obsługą CUDA:

pip uninstall -y torch pip install torch --index-url https://download.pytorch.org/whl/cu126

Sprawdź, czy GPU jest widoczne (ma wypisać True):

python -c "import torch; print(torch.cuda.is_available())"
Skrypt automatycznie wykryje GPU, pokaże jego nazwę i ilość VRAM, użyje go do transkrypcji (z fp16 dla szybkości), a gdyby zabrakło pamięci — sam przełączy się awaryjnie na CPU.
Dobór wersji CUDA i VRAM. Numer cu1XX dopasuj do swojego sterownika (sprawdź poleceniem nvidia-smi — pole „CUDA Version"); zwykle działa cu121, cu124, cu126 lub cu128. Model large-v3 potrzebuje ~10 GB VRAM — na kartach 8 GB (np. RTX 4060 Ti) lepszym wyborem jest turbo (jakość ~large-v3, mieści się w pamięci) lub medium.

📝 Gotowe polecenia (kopiuj-wklej)

W poleceniach zastąp ADRES linkiem do filmu, a ścieżki plików swoimi. Linki i ścieżki ze spacjami zawsze w cudzysłowach.

Pobieranie z YouTube

Film w MP4 (najlepsza jakość):
python .\youtube_downloader.py "ADRES"
Samo audio w MP3:
python .\youtube_downloader.py "ADRES" --format mp3
Film + napisy w języku oryginału (np. polski film):
python .\youtube_downloader.py "ADRES" --subs
Film obcojęzyczny + tłumaczenie napisów na polski (np. hiszpański):
python .\youtube_downloader.py "ADRES" --subs --source-lang es --translate-to pl
Nie wiesz, w jakim języku jest film? Auto-wykrywanie + tłumaczenie na PL:
python .\youtube_downloader.py "ADRES" --subs --source-lang auto --translate-to pl

Plik z dysku (bez pobierania)

Plik lokalny → napisy w oryginale:
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang en
Plik lokalny obcojęzyczny → tłumaczenie na polski:
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang pt --translate-to pl

Filmy z logowaniem i własny katalog wyników

Film 18+ / „potwierdź, że nie jesteś botem" (ciasteczka z przeglądarki):
python .\youtube_downloader.py "ADRES" --cookies-from-browser chrome
Zapisz wszystko do wybranego folderu:
python .\youtube_downloader.py "ADRES" --subs --output-dir "C:\pobrane"

Najwyższa jakość

GPU + szybki model wysokiej jakości (turbo) + nazwy własne w podpowiedzi:
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang pt --translate-to pl --model turbo --prompt "Pomni, Ragatha, Jax, Caine, Gangle, Kinger, Zooble"
Maksymalna dokładność (large-v3 — wymaga ~10 GB VRAM lub cierpliwości na CPU):
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang pt --translate-to pl --model large-v3

Napisy w pliku wideo

Wtop polskie napisy NA STAŁE w obraz (hardsub — działa wszędzie):
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang pt --translate-to pl --burn
Osadź przełączalną ścieżkę napisów (szybkie, bez kodowania):
python .\youtube_downloader.py --file "C:\filmy\wideo.mp4" --source-lang pt --translate-to pl --embed

🎯 Jak uzyskać najlepszą jakość

Dźwignie jakości, od najważniejszej:

  1. Poprawny język — to najczęstsza przyczyna złych napisów. Jeśli film jest dubbingowany (np. ma angielską czołówkę, a dialog po portugalsku), podaj język wprost: --source-lang pt. Whisper przy złym języku „transkrybuje" mowę fonetycznie i powstaje bełkot. Tryb auto próbkuje teraz kilka momentów, więc rzadziej się myli, ale jawne podanie języka jest najpewniejsze.
  2. Większy model — turbo lub large-v3 zamiast base.
  3. Podpowiedź --prompt — podaj imiona postaci, nazwy własne i terminologię (najlepiej w języku audio). Wtedy pisownia nazw jest poprawna.
  4. GPU — nie zmienia jakości, ale sprawia, że duże modele stają się praktyczne czasowo (patrz sekcja GPU).
Tłumaczenie pracuje na transkrypcji — jeśli transkrypcja jest zła (np. zły język), tłumaczenie też będzie złe. Najpierw zadbaj o dobrą transkrypcję.

📄 Pliki wynikowe i napisy SRT

Pliki powstają obok źródła. Przyrostek to kod języka, np. _PL, _EN, _PT. Przy tłumaczeniu zapisywane są obie wersje:

Tytuł_filmu.mp4 # pobrany film (lub Twój plik wejściowy) Tytuł_filmu_PT.srt # napisy w oryginale (tu: portugalski) Tytuł_filmu_PT.txt # pełna transkrypcja tekstowa (oryginał) Tytuł_filmu_PL.srt # napisy przetłumaczone (tu: polski) Tytuł_filmu_PL.txt # pełny tekst tłumaczenia

Jak użyć napisów SRT?

  • VLC: Napisy → Dodaj plik napisów → wybierz .srt
  • Automatycznie: trzymaj plik .srt w tym samym folderze i o tej samej nazwie co film — większość odtwarzaczy wczyta go sam

🔧 Rozwiązywanie problemów

„python nie jest rozpoznawany jako polecenie"

Użyj py zamiast python, albo zainstaluj Python ponownie z opcją „Add to PATH".

Pobieranie zawodzi: „HTTP Error 403" / „Unable to extract"

Skrypt sam próbuje wielu strategii, a przy takim błędzie automatycznie aktualizuje yt-dlp i ponawia (YouTube często zmienia coś po swojej stronie). Jeśli mimo to nie działa — sprawdź film w przeglądarce lub zaktualizuj ręcznie: pip install -U yt-dlp.

„Sign in to confirm your age" / „Confirm you're not a bot"

Wideo wymaga zalogowania. Skrypt ma do tego wbudowany parametr — przekaż przeglądarkę, z której jesteś zalogowany na YouTube:

python .\youtube_downloader.py "ADRES" --cookies-from-browser chrome

Działają też: firefox, edge, brave, opera.

„HTTP Error 429 Too Many Requests"

YouTube ogranicza liczbę żądań — odczekaj kilka minut lub zmień sieć / włącz VPN.

Blokada regionalna

Skrypt próbuje obejścia automatycznie (--geo-bypass). Jeśli nie pomoże — włącz VPN.

Napisy to bełkot / powtórzenia / zły język

  • Najczęściej zły wykryty język — podaj go wprost, np. --source-lang pt
  • Użyj większego modelu: --model turbo lub large-v3
  • Dodaj --prompt z nazwami własnymi

Brak pamięci VRAM (GPU) — „out of memory"

Skrypt sam przełączy się na CPU, ale lepiej wybrać mniejszy model: --model turbo lub medium. Zamknij też aplikacje zużywające VRAM (przeglądarki, gry, inne modele AI).

Whisper / transkrypcja bardzo wolna

Działasz na CPU. Zainstaluj PyTorch z CUDA (sekcja GPU) albo użyj mniejszego modelu (small/base).

💾 Kod skryptu

Skopiuj poniższy kod do pliku youtube_downloader.py:

import re import subprocess import os import argparse import sys import shutil # Wymuś UTF-8 na wyjściu konsoli (Windows) — inaczej polskie znaki (ł, ż...) # i emoji powodują UnicodeEncodeError przy domyślnym kodowaniu cp1252. try: sys.stdout.reconfigure(encoding='utf-8') sys.stderr.reconfigure(encoding='utf-8') except (AttributeError, ValueError): pass def extract_url(text): """Czyszczenie tekstu i wyciąganie linku YouTube""" clean_text = re.sub(r'[\[\]\(\)]', '', text).strip() pattern = r'(https?://(?:www\.)?youtube\.com/watch\?v=[0-9A-Za-z_-]{11})' match = re.search(pattern, clean_text) if match: return match.group(1) return clean_text def check_whisper_installed(): """Sprawdza czy Whisper jest zainstalowany""" try: import whisper return True, whisper.__version__ except ImportError: return False, None def check_ffmpeg_installed(): """Sprawdza czy ffmpeg jest zainstalowany""" return shutil.which('ffmpeg') is not None def check_translator_installed(): """Sprawdza czy biblioteka deep-translator jest zainstalowana""" try: import deep_translator # Human-readable language names for messages LANGUAGE_NAMES = { 'auto': 'auto-detect', 'pl': 'Polish', 'en': 'English', 'es': 'Spanish', 'de': 'German', 'fr': 'French', 'it': 'Italian', 'pt': 'Portuguese', 'ru': 'Russian', 'uk': 'Ukrainian', 'cs': 'Czech', 'sk': 'Slovak', 'nl': 'Dutch', 'ja': 'Japanese', 'zh': 'Chinese', 'ko': 'Korean', 'ar': 'Arabic', 'tr': 'Turkish', 'sv': 'Swedish', 'no': 'Norwegian', 'da': 'Danish', 'fi': 'Finnish', 'hu': 'Hungarian', 'ro': 'Romanian', 'el': 'Greek', } def lang_name(code): """Zwraca czytelną nazwę języka dla podanego kodu""" return LANGUAGE_NAMES.get(code, code) def translate_texts(texts, target_lang, source_lang='auto', max_workers=8): """ Tłumaczy listę tekstów na język docelowy używając Google Translate. Segmenty tłumaczone są RÓWNOLEGLE (pula wątków) — przy długich filmach to wielokrotne przyspieszenie względem tłumaczenia jeden po drugim, a kolejność i przyporządkowanie do znaczników czasu pozostają dokładne (mapowanie 1:1). Jeśli pojedynczy segment się nie przetłumaczy, zostaje oryginalny tekst. Zwraca listę przetłumaczonych tekstów (tej samej długości co wejście). """ from concurrent.futures import ThreadPoolExecutor from deep_translator import GoogleTranslator src = source_lang or 'auto' # Google używa 'auto' do auto-wykrywania źródła total = len(texts) def work(item): idx, text = item original = text.strip() if not original: return idx, text try: # Nowy translator na wywołanie — bezpieczne wątkowo (brak współdzielonego stanu) result = GoogleTranslator(source=src, target=target_lang).translate(original) return idx, (result if result else original) except Exception: return idx, original # silently keep the original — don't abort everything translated = [None] * total completed = 0 # Network I/O releases the GIL, so threads give a real speedup with ThreadPoolExecutor(max_workers=max_workers) as executor: for idx, text in executor.map(work, enumerate(texts)): translated[idx] = text completed += 1 if completed % 25 == 0 or completed == total: print(f' ...translated {completed}/{total} segments') return translated def format_timestamp_vtt(seconds): """Konwertuje sekundy na format WebVTT (00:00:00.000 — kropka zamiast przecinka).""" return format_timestamp_srt(seconds).replace(',', '.') def write_vtt(vtt_file, segments, texts=None): """ Zapisuje plik napisów w formacie WebVTT (.vtt) — używany m.in. w odtwarzaczach WWW. Różni się od SRT nagłówkiem 'WEBVTT' i kropką w znaczniku czasu. """ with open(vtt_file, 'w', encoding='utf-8') as f: f.write('WEBVTT\n\n') for i, segment in enumerate(segments, 1): start = format_timestamp_vtt(segment['start']) end = format_timestamp_vtt(segment['end']) text = (texts[i - 1] if texts is not None else segment['text']).strip() f.write(f'{start} --> {end}\n') f.write(f'{text}\n\n') def write_srt(srt_file, segments, texts=None): """ Zapisuje plik napisów SRT. - segments: lista segmentów Whisper (zawierają 'start', 'end', 'text') - texts: opcjonalna lista tekstów nadpisująca 'text' z segmentów (używana dla wersji przetłumaczonej) """ with open(srt_file, 'w', encoding='utf-8') as f: for i, segment in enumerate(segments, 1): start = format_timestamp_srt(segment['start']) end = format_timestamp_srt(segment['end']) text = (texts[i - 1] if texts is not None else segment['text']).strip() f.write(f'{i}\n') f.write(f'{start} --> {end}\n') f.write(f'{text}\n\n') def show_whisper_installation_guide(): """Wyświetla instrukcję instalacji Whisper""" print('\n' + '=' * 70) print('⚠️ WHISPER IS NOT INSTALLED') print('=' * 70) print('\nWhisper is a free OpenAI AI model for audio transcription.') print('It runs locally (offline) and handles many languages very well!\n') print('📦 INSTALL WHISPER:\n') print('1. Install Whisper:') print(' pip install -U openai-whisper\n') print('2. Install ffmpeg (if you do not have it yet):\n') print(' Linux/Ubuntu:') print(' sudo apt update && sudo apt install ffmpeg\n') print(' Windows:') print(' - Download from: https://ffmpeg.org/download.html') print(' - Unpack and add to PATH\n') print(' macOS:') print(' brew install ffmpeg\n') print('3. After installing, run the script again with the --subs option\n') print('=' * 70) def detect_language_robust(model, audio_path, fractions=(0.15, 0.4, 0.65)): """ Wykrywa język mowy próbkując KILKA momentów nagrania (a nie tylko pierwsze 30 s, jak domyślnie Whisper). Dzięki temu czołówka/intro w innym języku nie zafałszowuje wyniku. Zwraca: (kod_języka, pewność 0..1).
